# -Student-Performance-Predictor

## 📌 Overview

This project predicts a student's final academic performance based on factors like study hours, attendance, sleep hours, and previous marks using Machine Learning.

## 🎯 Objective

To analyze student-related factors and predict final scores using AI/ML concepts.

## ⚙️ Features

* Predicts final score
* Uses regression model
* Simple user input system
* Easy to understand and run

## 🧠 Concepts Used

* Supervised Learning
* Linear Regression
* Feature Selection
* Model Evaluation

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn

## ▶️ How to Run

1. Install libraries:
   pip install pandas scikit-learn

2. Run:
   python main.py

## 💡 Example

Input:
Study Hours = 5
Attendance = 90
Sleep Hours = 7
Previous Marks = 75

Output:
Predicted Final Score: 80.4
